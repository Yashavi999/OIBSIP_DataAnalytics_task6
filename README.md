# Wine Quality Prediction Using Machine Learning

## Project Overview

This project aims to predict **wine quality** by analyzing its chemical characteristics. Various machine learning classification algorithms are applied to study the dataset and accurately classify the quality of wine.

## Dataset

WineQT.csv

The dataset includes several chemical properties of wine such as acidity levels, density, alcohol percentage, and pH values, which are used to determine wine quality ratings.

## Objectives

* Explore and analyze the wine quality dataset
* Perform data preprocessing and normalization
* Develop multiple classification models
* Compare the effectiveness of different models
* Visualize evaluation results and performance metrics

## Models Used

* Random Forest Classifier
* Support Vector Classifier (SVC)
* Stochastic Gradient Descent (SGD) Classifier

## Steps Performed

1. Loaded and explored the dataset
2. Checked for missing or inconsistent data
3. Performed exploratory data visualization
4. Separated input features and target variable
5. Split the dataset into training and testing sets
6. Applied feature scaling using StandardScaler
7. Trained models and generated predictions
8. Evaluated model performance using accuracy, confusion matrix, and classification report

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

## Results

Out of all the implemented models, the Random Forest classifier achieved the highest accuracy and produced the most reliable classification results.

## Conclusion

This project demonstrates how machine learning classification algorithms can be applied to predict wine quality. It also emphasizes the importance of preprocessing, feature scaling, model comparison, and proper evaluation when working with real-world datasets.
